## 🦖 Git Commit Message
|*Type*|*Subject*|
|:---|:---|
|**[PLATFORM]**|문제 해결|
|**[FIX]**|수정|
|**[DOCS]**|문서 (문서 추가, 수정, 삭제)|

ex) [PLATFORM] 문제번호. 문제이름 `git commit -m '[BOJ] 1001. A+B'`
